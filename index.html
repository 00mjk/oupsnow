oupsnow by shingara

A bugtracker in Rails/MongoMapper

A Bug tracker multi project, simply, but completed like bug tracking. It's written in Rails with MongoDB like database

Dependencies

* Ruby of 1.8.6 or greater * Rails 2.3.5 * MongoMapper 0.6.4 * A MongoDB 1.0.1 or greater

Install

With the tar.gz or any other archive: * Extract sources to a folder * Create a database.yml file in the config directory.
